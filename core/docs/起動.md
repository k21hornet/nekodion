# Nekodion Backend

> [!NOTE]
> 各設定値は開発メンバーに聞いてください。

## 起動方法

### 準備

1. IntelliJ > 実行 > 構成の編集を開く
2. 新規構成の追加でGradleを選択する

### API

| 項目                | 設定値                                                         |
| ------------------- | -------------------------------------------------------------- |
| 実行 > タスクと引数 | `api:bootRun`                                                  |
| Gradleプロジェクト  | `core`                                                         |
| 環境変数            | `DB_NAME=nekodion;DB_USERNAME=;DB_PASSWORD=;ISSUER=;AUDIENCE=` |

### Batch

| 項目                | 設定値                                       |
| ------------------- | -------------------------------------------- |
| 実行 > タスクと引数 | `batch:bootRun --args='--batch.execute='`    |
| Gradleプロジェクト  | `core`                                       |
| 環境変数            | `DB_NAME=nekodion;DB_USERNAME=;DB_PASSWORD=` |

- batch.execute

  | バッチ             | 引数       | 備考             |
  | ------------------ | ---------- | ---------------- |
  | Hello Worldバッチ  | HelloWorld | バッチ起動テスト |
  | Gmail Importバッチ | gmail      | 入出金記録バッチ |
